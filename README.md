Java Mini Project Collection 

📌 Overview

This repository contains 5 Java-based projects focused on problem-solving,
pattern analysis, and real-world application logic using core Java concepts.

🚀 Project List

1. Intelligent Number Series Solver Using Alternating Pattern Analysis

Description:
A Java-based system that analyzes number sequences and predicts the next value by identifying alternating and mathematical patterns.
Features:
* Detects arithmetic and geometric patterns
* Supports alternating operations (+, −, ×, ÷)
* Difference-based sequence analysis
* Predicts next number in series
